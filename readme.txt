Тестовое задание выполнено на Python.
В проекте используются исключительно встроенные библиотеки для упрощения тестирования.

Перечень команд по ТЗ для используемого языка.
python3 main.py 1
python3 main.py 2
python3 main.py 3
python3 main.py 4
python3 main.py 5
python3 main.py 6

Последней командой "python3 main.py 6" можно провести оптимизацию БД,
после чего снова выполнить замеры "python3 main.py 5"
